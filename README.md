# Flappy-Fish-Game
Flappy Fish is a 2D side-scrolling game where players control a fish, navigating through underwater rocks while collecting coins. The game features physics-based movement, requiring precise timing to avoid obstacles. With smooth animations, a high-score system, and pause/resume options, it delivers a fun and challenging underwaterÂ adventure.
